# 基于Angular9构建的SPA后台管理系统  
##演示地址:<https://leemagination.github.io/ems-web/>
该项目是使用Angular9和 ng-zorro-antd 构建的一款单页面应用后台管理系统。  
系统具有登录、注册、多语言切换、权限控制等功能，页面之间通过tab的方式进行切换。

>不支持移动端  
>该项目演示中的数据由mockjs生成   
>待办项和复杂表格页的内容是保存在localStorage，可随意增删改查。
